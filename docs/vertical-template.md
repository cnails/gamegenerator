# Шаблон Vertical 9:16

Вертикальная заготовка (`GameTemplate.VERTICAL_STANDARD`) фиксирует портретное соотношение 9:16, автоматически организует безопасную область и реализует типовой поток вертикального шутера. Файл сцены: `src/scenes/templates/VerticalStandardScene.ts`.

## Быстрый старт
- Выберите шаблон **«Вертикальный 9:16»** (или задайте `template: "verticalStandard"` в `GameConfig`).
- Настройте параметры `scrollSpeed`, `spawnDensity`, `roundLength`, опционально `health`.
- Запустите игру — `GameManager` подключит `VerticalStandardScene` и обеспечит resize/cleanup.
- Для кастомизации расширяйте `spawnObstaclePattern`, `spawnCollectibleCluster`, `spawnEnemy` или добавляйте собственные пайплайны спавна.

## Параметры

| Параметр | Диапазон | Назначение |
| --- | --- | --- |
| `scrollSpeed` | 160–420 | Скорость вертикального авто-скролла и падения объектов |
| `spawnDensity` | 0.5–2.5 | Множитель частоты появления препятствий, врагов и бонусов |
| `roundLength` | 45–240 | Длительность раунда (таймер и прогресс-бар в HUD) |
| `health` *(опц.)* | 2–6 | Количество жизней игрока |

## Архитектурные блоки
- **Safe-area 9:16**: `safeViewport` + `playBounds` сохраняют портретное окно при любом размере канваса.
- **Ввод**: управление в одной оси (`Left/Right` и pointer/touch) с плавным следованием и клампом по `playBounds`.
- **Потоки событий**: единый `handleSpawns` распределяет таймеры препятствий, бонусов и врагов; легко добавить новые очереди.
- **HUD**: счет, здоровье, таймер и прогресс-бар обновляются синхронно с длительностью раунда.

## Кастомизация
- Используйте `this.getVisualColors()` для автоматической темы или подставьте свои текстуры через `ensure*Texture`.
- Расширяйте игровой цикл, переопределяя `handleSpawns`, `updateEnemiesBehavior`, `applyDamage`.
- Добавляя новые группы столкновений, не забудьте зарегистрировать их в `registerCollisions`.
- Для нестандартных UI можно заменить `createHud` и `updateProgressBar`, сохранив базовый API.
